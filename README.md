\# 基于YOLO11的变电站设备缺陷检测系统

本项目针对**电力设备小目标缺陷、类别样本不均衡**问题，基于YOLO11轻量化模型实现变电站多类缺陷智能检测，适用于电力巡检智能化场景。

项目完成数据集扩充、超参数调优、小目标增强优化，最终模型检测精度显著提升，满足实际检测需求。

## 📌 项目简介

电力设备缺陷存在**目标尺寸小、缺陷特征弱、类别分布极不均衡**等特点，常规检测模型漏检、误检严重。

本项目基于 YOLO11n / YOLO11s 模型，针对电力场景做专项优化：

- 通过 Roboflow 平台**扩充少数类缺陷样本**，缓解类别不平衡问题

- 针对性调优损失权重、学习率、数据增强策略，强化小目标缺陷学习

- 适配短轮次高效训练，在有限迭代内完成模型收敛与精度提升

- 最终 mAP50 由 0\.618 提升至 0\.70\+，有效改善绝缘子缺陷等小目标漏检问题

## ⚙️ 核心调优策略（项目创新点）

- **数据层面**：Roboflow 针对性扩充少数类缺陷样本，不删除原始数据，最大化保留缺陷特征

- **损失权重优化**：提升分类权重、框回归权重，强制模型关注小目标与长尾缺陷

- **学习率优化**：降低初始学习率、延长预热期，避免多数类主导训练

- **轻量化增强**：降低 Mosaic 增强强度，开启 Copy\-Paste 小目标增强，避免裁剪破坏细微缺陷

- **训练策略**：适配30轮短周期训练，提前关闭增强保证后期收敛精度
